## 📰 Key Highlights

> H Company officially released the Holo 3.1 computer control AI agent model series on June 2, 2026, offering four specifications: 0.8B ultra-lightweight local version, 4B low-cost version, 9B performance-balanced version, and flagship 35B-A3B.

On the mobile automation benchmark AndroidWorld, the 35B-A3B model score jumped from 67% in the previous generation to 79.3%, while the 4B and 9B models rose from 58% to 72% — a pretty significant leap. When integrated with the official product framework Holotab, overall performance improved by over 25% compared to Holo 3, and native support for function-calling was added, dramatically reducing performance gaps across different deployment environments.

For quantized inference, the official release also includes compressed weights in three formats: FP8, Q4 GGUF, and NVFP4. On NVIDIA DGX Spark hardware, NVFP4 delivers 1.74x higher throughput compared to BF16 full precision. After comprehensive optimization, inference time per step shrank from 6.8 seconds to 3.3 seconds — roughly 2x faster — with accuracy loss on OSWorld tests being only about 2 points, virtually imperceptible.

Deployment options support both cloud API and local execution. The local mode works on Windows and Mac, keeping data within the user's own network for a balance of performance and privacy protection. The entire model series is now available for download on Hugging Face and can be accessed via H Company's official API for cloud calls.
